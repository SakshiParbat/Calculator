# Calculator App 🧮

A responsive calculator application built using HTML, CSS, and JavaScript. The project performs basic arithmetic operations through an intuitive user interface and demonstrates DOM manipulation, event handling, and JavaScript logic implementation.

## 🚀 Features

* Addition, Subtraction, Multiplication, and Division
* Responsive Design
* Interactive User Interface
* Real-time Calculation
* Clear and Reset Functionality
* Keyboard-Friendly Operations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

Calculator/
├── index.html
├── style.css
├── script.js

## 🎯 Learning Outcomes

* DOM Manipulation
* Event Handling
* JavaScript Functions
* UI Development
* Responsive Design Principles

## ▶️ How to Run

1. Clone the repository.
2. Open the project folder.
3. Run `index.html` in any web browser.
